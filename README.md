# dossier-EST-2025

Dossier en formato HTML para EST 2025. Este repositorio contiene los archivos estáticos necesarios para visualizar el dossier localmente o desplegarlo en GitHub Pages.

## Estado
✅ COMPLETO — Última actualización: 2026-02-24

## Descripción corta
Dossier estático en HTML que reúne la documentación y el material del proyecto EST 2025. El proyecto está listo para su visualización y despliegue; no contiene imágenes embebidas.

## Características
- Documentos en HTML listos para abrir en el navegador.
- Código estático ligero y fácil de servir.
- Preparado para despliegue en GitHub Pages.

## Tecnologías
- HTML (100%)

## Requisitos
- Navegador web moderno (Chrome, Firefox, Edge, Safari).
- Para servir localmente (opcional): Python 3 o npx live-server.

## Instalación y uso
1. Clonar el repositorio

```bash
git clone https://github.com/Akxlarre/dossier-EST-2025.git
cd dossier-EST-2025
```

2. Visualizar localmente
- Abrir `index.html` directamente en el navegador.
- O servir con Python:

```bash
python3 -m http.server 8000
# luego abrir http://localhost:8000
```

- O usar live-server (Node.js):

```bash
npx live-server
```

## Despliegue (GitHub Pages)
1. En el repositorio en GitHub, ir a Settings → Pages.
2. Seleccionar la rama `main` y la carpeta `/ (root)` o `/docs` según donde esté `index.html`.
3. Guardar. La URL pública será: `https://Akxlarre.github.io/dossier-EST-2025`

Alternativa: crear una rama `gh-pages` con los archivos estáticos.
